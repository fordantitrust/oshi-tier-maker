📖 คู่มือการใช้งาน

📁 อัพโหลดรูป

  • คลิกหรือลากไฟล์มาวางที่ช่อง upload
  • รองรับ JPG · PNG · GIF · WebP
  • อัพโหลดหลายไฟล์พร้อมกันได้
📌 รูปทั้งหมดเก็บไว้ในเบราว์เซอร์ของคุณ ไม่ถูกส่งไปที่ใดทั้งสิ้น

🖱️ จัดรูปลง Tier

  • ลากรูปจาก pool ด้านบนลงใน tier ที่ต้องการ
  • ลากย้ายระหว่าง tier ได้อิสระ หรือลากกลับ pool
  • กด × บนรูปเพื่อลบออก

✏️ จัดการ Tier

  • คลิกชื่อ tier → พิมพ์ชื่อใหม่ → กด Enter หรือคลิกออกเพื่อบันทึก
  • คลิกพื้นหลังสี → เลือกสีจาก 8 สี
  • ปุ่ม × มุมขวาบน (เมาส์ชี้เพื่อแสดง) → ลบ tier นั้น
  • เมื่อลบ tier รูปในนั้นจะกลับมาที่ pool อัตโนมัติ
  • ปุ่ม ＋ เพิ่ม tier → เพิ่ม tier ใหม่ พร้อมตั้งชื่อทันที

🎨 สร้างภาพ PNG

  • เลือก รูปต่อแถว (3–8) ให้เหมาะกับจำนวนรูปที่มี
  • กด "สร้างภาพ PNG" → ดาวน์โหลด oshi-tier.png ทันที
  • Tier ที่ไม่มีรูปจะไม่ปรากฏในภาพ

📦 Export / Import

  • Export → บันทึกรูปทั้งหมดและการจัด tier เป็นไฟล์ .zip
  • Import → โหลดไฟล์ .zip เพื่อกู้คืนหรือย้ายข้อมูลจากเครื่องอื่น
💡 ควร Export ไว้ก่อน เนื่องจากรูปเก็บในเบราว์เซอร์ — ถ้าล้างข้อมูลเบราว์เซอร์หรือใช้โหมด Private รูปจะหายทั้งหมด

🗄️ พื้นที่จัดเก็บ

  • รูปทั้งหมดเก็บไว้ในเบราว์เซอร์ — ดูพื้นที่ที่ใช้ได้จาก progress bar ด้านบน
  • จำกัดพื้นที่รวม 50 MB — จะแจ้งเตือนเมื่อใกล้เต็ม

💾 บันทึกอัตโนมัติ

  • ทุกการเปลี่ยนแปลงจะถูกบันทึกโดยอัตโนมัติ
  • เปิดหน้าใหม่หรือ refresh — tier และตำแหน่งรูปกลับมาเหมือนเดิม
  • กด "ล้างทั้งหมด" เพื่อล้างข้อมูลและเริ่มต้นใหม่
